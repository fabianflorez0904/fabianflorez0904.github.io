# Portafolio Personal - Fabian Florez

Este es mi portafolio personal, construido con HTML, CSS y Bootstrap. El sitio es completamente responsivo y está optimizado para ser desplegado en GitHub Pages.

## Estructura del Proyecto

```
.
├── index.html          # Página principal
├── styles.css         # Estilos personalizados
├── assets/           # Directorio de imágenes
│   ├── profile.jpg   # Foto de perfil
│   └── hero-bg.jpg   # Imagen de fondo del hero
└── README.md         # Este archivo
```

## Requisitos de Imágenes

Para que el sitio funcione correctamente, necesitas agregar las siguientes imágenes en el directorio `assets/`:

1. `profile.jpg`: Una foto de perfil profesional en formato JPG
   - Resolución recomendada: 800x800 píxeles
   - Tamaño máximo: 500KB
   - Formato: JPG

2. `hero-bg.jpg`: Una imagen de fondo para la sección hero
   - Resolución recomendada: 1920x1080 píxeles
   - Tamaño máximo: 1MB
   - Formato: JPG
   - Debe ser una imagen que represente tecnología o desarrollo de software

## Despliegue en GitHub Pages

1. Crea un nuevo repositorio en GitHub
2. Sube todos los archivos a tu repositorio
3. Ve a la configuración del repositorio (Settings)
4. Desplázate hasta la sección "GitHub Pages"
5. En "Source", selecciona la rama "main" o "master"
6. Haz clic en "Save"

Tu sitio estará disponible en: `https://[tu-usuario].github.io/[nombre-del-repositorio]`

## Personalización

Puedes personalizar el sitio modificando:

- `index.html`: Para cambiar el contenido y la estructura
- `styles.css`: Para modificar los estilos y la apariencia
- Imágenes en el directorio `assets/`: Para actualizar las imágenes
